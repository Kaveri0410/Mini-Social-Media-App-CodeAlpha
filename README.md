# Social Media Platform – CodeAlpha Internship Task 2

## Project Description
This is a backend-based **Social Media Platform** developed as part of **CodeAlpha Internship – Task 2**.  
The project focuses on RESTful API development and testing using Postman.

---

## Features
- User Registration API
- User Login API
- Create Post API
- Follow User API
- MongoDB Atlas database integration

---

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Postman

---

## API Endpoints Tested

### Authentication APIs
- POST `/api/auth/register`
- POST `/api/auth/login`

### Post API
- POST `/api/posts/create`

### Follow API
- POST `/api/follow/follow`

---

## How to Run the Project

1. Install dependencies:npm install

2. Start server:node backend/server.js

3. Server will run on:http://localhost:5000
